🍔 Diamond Fry Center - Food Delivery App
A complete MERN stack food delivery application designed to manage a restaurant's digital menu, customer orders, and admin operations.

🚀 Key Features
Customer Module:

Browse food menu with categories (Fast food, Pizza, etc.).

Add items to cart and manage quantities.

Checkout and track real-time order status.

User authentication (Login/Register/Forgot Password).

Admin Module:

Manage Items: Add, edit, or delete food items from the menu.

Order Management: Track and update customer orders.

Customer Management: View and manage user profiles.

🛠 Tech Stack
Frontend: React.js, Vite, Tailwind CSS, Redux (State Management).

Backend: Node.js, Express.js.

Database: MongoDB.

Deployment: Vercel.

📋 Topics/Modules
Authentication: Secure Login/Signup system.

Product Management: Admin-only access to modify the menu.

Shopping Cart: Persistent cart management using Redux.

Order Workflow: Seamless transition from browsing to ordering.

Responsive UI: Mobile-friendly design using Tailwind CSS.
